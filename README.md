# Напір

Веб-калькулятор тиску води в приватному будинку зі свердловиною. Клієнтський застосунок без бекенду — рахує гідравліку (втрати на тертя, статичний підйом, баланс напору) і показує, який тиск буде в крані. Розгортається як статика на GitHub Pages.

**Стек:** Svelte + Svelte Flow + Vite, розрахунковий рушій на чистому TypeScript (з юніт-тестами Vitest).
