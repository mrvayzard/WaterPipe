# Напір

Веб-калькулятор тиску води в приватному будинку зі свердловиною. Клієнтський застосунок без бекенду — рахує гідравліку (втрати на тертя, статичний підйом, баланс напору) і показує, який тиск буде в крані. Розгортається як статика на GitHub Pages.

**Стек:** Svelte + Svelte Flow + Vite, розрахунковий рушій на чистому TypeScript (з юніт-тестами Vitest).

## Документи

- [Технічне завдання](napir-tz.md) — фізична модель, функціональні та нефункціональні вимоги.
- [План впровадження](napir-plan.md) — етапи реалізації v1.

## Статус

🚧 Етап 1 — розрахунковий рушій (`src/engine/`): формули (Хазен–Вільямс + Дарсі–Вейсбах-оракул), граф-схема, баланс напору, тести Vitest. Далі — UI поверх рушія (Етап 2).

Прототип фізики: `napir.html` (v0.1).
